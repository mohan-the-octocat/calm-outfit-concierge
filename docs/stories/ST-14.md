# ST-14 Cloud Run + CI/CD

## Description
Set up Dockerfile, Cloud Build trigger, Artifact Registry, and Cloud Run deployment for API services; manage env/secrets via Secret Manager.

## Expected Result
Push to main builds and deploys to target Cloud Run service with correct env vars/secrets injected; health checks pass.

## Definition of Done
- [ ] Dockerfile builds locally; minimal image.
- [ ] Cloud Build trigger configured on main; pushes to Artifact Registry.
- [ ] Cloud Run service deployed with autoscaling defaults and min=0.
- [ ] Secret Manager wired for OpenAI key and other creds; no secrets in repo.
- [ ] Health check endpoint responds 200; deployment verified post-build.
- [ ] CI pipeline status visible; docs on how to deploy locally and via CI.
