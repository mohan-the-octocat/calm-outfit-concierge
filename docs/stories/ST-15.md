# ST-15 Networking (VPC/Armor)

## Description
Configure Serverless VPC Access and Cloud NAT for outbound traffic; optional Cloud Armor policy for public endpoints.

## Expected Result
Cloud Run can reach OpenAI and databases through controlled egress; DB not publicly accessible; Armor rule applied if enabled.

## Definition of Done
- [ ] Serverless VPC Access connector created and attached to Cloud Run.
- [ ] Cloud NAT configured for outbound internet with minimal IPs.
- [ ] DB (AlloyDB/Cloud SQL) private IP only; no public ingress.
- [ ] (Optional) Cloud Armor policy created and bound to frontend/API if needed; basic rate limit rule.
- [ ] Connectivity test to OpenAI and DB passes through VPC.
- [ ] Network diagram/docs updated.
