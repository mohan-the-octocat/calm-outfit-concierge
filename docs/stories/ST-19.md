# ST-19 Image Handling Policy

## Description
Define and implement image handling: signed URL upload, transient processing, no raw image logging, bucket lifecycle rules.

## Expected Result
Images are processed via signed URLs; not stored beyond needed window; logs free of raw image data; lifecycle rules enforce retention.

## Definition of Done
- [ ] Signed URL flow documented and implemented (upload + read).
- [ ] No raw image bodies in logs; logging filters applied.
- [ ] GCS bucket lifecycle rule set for expiration.
- [ ] Security review notes recorded.
- [ ] Test upload/download path and verify lifecycle config.
