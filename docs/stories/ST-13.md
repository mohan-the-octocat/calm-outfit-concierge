# ST-13 Shipping SLA Loader

## Description
Load shipping SLA data into the SLA table (cut-offs, ETA ranges, carrier/method metadata) on a schedule.

## Expected Result
SLA table is current; bad rows rejected with logs; timezone info preserved; consumers can query by region/method.

## Definition of Done
- [ ] Schema for SLA table created with timezone-aware cut-off.
- [ ] Loader validates rows and rejects malformed entries.
- [ ] Scheduled run via Cloud Scheduler + Cloud Run Job.
- [ ] Metrics/logs for rows loaded/failed.
- [ ] Tests with multiple timezones and missing fields.
